# Power BI Sales Dashboard

## Overview

This project is an interactive **Power BI Sales Dashboard** built to analyze and visualize sales performance. It tracks key business metrics such as total sales, quantity, transactions, and average value, helping identify trends and performance insights.

## Objectives

* Monitor overall sales performance
* Analyze product/model-wise sales
* Track customer ratings and transaction behavior
* Identify sales trends by day and year
* Understand payment method distribution

## Key Features

* Data cleaning and transformation using **Power Query**
* KPI creation using **DAX measures**
* Interactive dashboard with **slicers and filters**
* Sales trend and yearly performance analysis
* Model-wise and brand-wise sales comparison
* Payment method distribution visualization
* Customer ratings analysis

## KPIs Included

* Total Sales
* Total Quantity Sold
* Total Transactions
* Average Sales Value

## Tools & Technologies

* Microsoft Power BI
* Power Query Editor
* DAX (Data Analysis Expressions)

## Dashboard Insights

* Identified top-performing mobile models and brands
* Analyzed yearly sales quantity trend
* Compared payment methods usage (UPI, Card, Cash, etc.)
* Evaluated customer rating distribution
* Observed daily sales performance pattern

## How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Microsoft Power BI Desktop**
3. Use slicers and filters to explore the dashboard
